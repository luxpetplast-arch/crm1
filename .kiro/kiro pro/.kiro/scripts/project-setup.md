# Project Setup Script

Automated project initialization with Kiro AI agents.

## Usage

```bash
# Run this for new projects
@project-supervisor Yangi loyiha setup qil
```

## Setup Steps

### 1. Choose Preset
```bash
# Select project type
- Startup MVP
- SaaS
- E-commerce
- Enterprise

# Load preset
@feature-manager Preset ni yuklash
```

### 2. Initialize Structure
```bash
@code-generator Project strukturasini yarat
- src/
- tests/
- docs/
- .kiro/
```

### 3. Setup Configuration
```bash
# Package.json
@dependency-manager Dependencies ni sozlash

# TypeScript/ESLint
@code-quality-enforcer Linting sozlash

# Git hooks
@git-commit-helper Pre-commit hooks
```

### 4. Database Setup
```bash
@database-optimizer Database schema yarat
@database-migration-manager Initial migration
```

### 5. CI/CD Setup
```bash
@ci-cd-specialist Pipeline yarat
- Build
- Test
- Deploy
```

### 6. Monitoring Setup
```bash
@monitoring-setup-specialist Monitoring sozlash
- Metrics
- Logs
- Alerts
```

### 7. Documentation
```bash
@documentation-writer Initial docs yarat
- README.md
- CONTRIBUTING.md
- API docs
```

## Post-Setup

```bash
@project-supervisor Setup verification
- All agents active
- All hooks configured
- Quality gates set
- CI/CD working
```
