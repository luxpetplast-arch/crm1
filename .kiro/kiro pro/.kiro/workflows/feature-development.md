# Feature Development Workflow

## 1. Planning Phase

```bash
# Step 1: Requirements
@project-supervisor Requirements ni tahlil qil va tasdiqlash

# Step 2: Design
@api-designer API dizaynni yarat
@react-specialist Frontend dizaynni ko'rib chiq
```

## 2. Implementation Phase

```bash
# Step 3: Database
@database-optimizer Schema o'zgarishlarini rejalashtir
@database-migration-manager Migration yarat

# Step 4: Backend
@code-generator API endpoints yarat
@nodejs-backend-specialist Backend kodini yoz

# Step 5: Frontend
@react-specialist Components yarat
@frontend-optimizer Performance optimizatsiya
```

## 3. Testing Phase

```bash
# Step 6: Tests
@unit-test-generator Unit tests yoz
@integration-tester Integration tests yoz
@regression-tester Regression test o'tkazish
@e2e-tester E2E tests yoz
```

## 4. Quality Assurance

```bash
# Step 7: Quality Checks
@code-reviewer Kod review qil
@security-auditor Xavfsizlik tekshir
@performance-optimizer Performance test
@accessibility-checker Accessibility tekshir
@code-quality-enforcer Quality gates tekshir
```

## 5. Documentation

```bash
# Step 8: Documentation
@documentation-writer API docs yoz
@documentation-writer User guide yarat
```

## 6. Deployment

```bash
# Step 9: Pre-deployment
@feature-manager Feature registry yangilash
@monitoring-setup-specialist Monitoring sozlash
@project-supervisor Pre-deployment check

# Step 10: Deploy
@ci-cd-specialist Staging ga deploy
# User acceptance testing
@ci-cd-specialist Production ga deploy
```

## 7. Post-deployment

```bash
# Step 11: Monitoring
@log-analyzer Loglarni tekshir
@metrics-collector Metricsni kuzatish
@alert-manager Alertlarni sozlash
```

## Checklist

- [ ] Requirements documented
- [ ] Design approved
- [ ] Database migrated
- [ ] Backend implemented
- [ ] Frontend implemented
- [ ] Tests written (80%+ coverage)
- [ ] Code reviewed
- [ ] Security audited
- [ ] Performance validated
- [ ] Documentation complete
- [ ] Deployed to staging
- [ ] User acceptance passed
- [ ] Deployed to production
- [ ] Monitoring active
