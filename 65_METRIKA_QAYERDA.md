## рҹ”Қ Agar Ko'rinmasa?

### Tekshirish Ro'yxati:

1. вң… **Server ishlaydimi?**
   ```bash
   # Terminal 1
   npm run dev
   ```

2. вң… **Backend ishlaydimi?**
   ```bash
   # Terminal 2
   cd server
   npm run dev
   ```

3. вң… **API ishlayaptimi?**
   ```bash
   node test-65-metrics.cjs
   ```

4. вң… **Browser Console tekshiring**
   ```
   F12 вҶ’ Console вҶ’ Xatolarni ko'ring
   ```

5. вң… **Network Tab tekshiring**
   ```
   F12 вҶ’ Network вҶ’ /api/statistics/business-metrics
   ```
