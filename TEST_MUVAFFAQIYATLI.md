## 🚀 Ishga Tushirish

### Server ishga tushirish
```bash
npm run dev
```

### Testlarni ishga tushirish

#### UI Mode (Tavsiya!)
```bash
npm run test:e2e:ui
```

#### Oddiy rejim
```bash
npm run test:e2e
```

#### Brauzerda ko'rish
```bash

```
