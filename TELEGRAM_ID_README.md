# 🆔 Telegram ID Tizimi - Tezkor Qo'llanma

## 🚀 Tezkor Boshlash

### Mijoz Uchun (30 soniya)

1. Telegram botni oching
2. `/start` yuboring
3. ID'ni oling: `A1B2C3D4`
4. Admin bilan baham ko'ring

### Admin Uchun (1 daqiqa)

1. Saytda "Mijoz Qo'shish" tugmasini bosing
2. "Telegram ID" maydoniga ID ni kiriting: `A1B2C3D4`
3. Boshqa ma'lumotlarni to'ldiring
4. "Mijoz Yaratish" tugmasini bosing

### ✅ Tayyor!

Mijoz endi:
- 🛒 Telegram orqali buyurtma bera oladi
- 💰 Balansini ko'ra oladi
- 📊 Sotuvlar tarixini ko'ra oladi
- 🔔 Bildirishnomalar oladi
